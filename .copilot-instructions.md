## Project Structure Notes

- `assets/Script/Controller/` - Game controllers (GameController, LevelSelectController, etc.)
- `assets/Script/Model/` - Data models and game logic
- `assets/Script/View/` - Visual components (GridView, CellView, EffectLayer)
- `assets/resources/` - Runtime loadable resources
- `openemoji/` - Source emoji PNG files (not included in build)

## Cocos Creator 3.x Notes

- Using TypeScript with strict type checking
- Cocos Creator version: 3.8.6
