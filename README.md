# 开心消消乐
使用 Cocos Creator 编写的三消游戏。

**最新版本：Cocos Creator 3.8.6 + TypeScript**

# 更新记录

2018/07/01 增加一些音效  
2019/01/30 升级工程到 `cocos creator` `v2.0.7`  
2024/12/27 **升级工程到 `cocos creator` `v3.8.6` + 迁移到 TypeScript**

# 迁移说明

项目已从 Cocos Creator 2.4.3 (JavaScript) 迁移到 Cocos Creator 3.8.6 (TypeScript)。

**重要提示：** 代码迁移已完成，但场景文件 (.fire) 需要在 Cocos Creator 3.8.6 编辑器中打开和转换。

详细迁移指南请查看：[MIGRATION_GUIDE.md](./MIGRATION_GUIDE.md)

## 主要改进

- ✅ 全部代码迁移到 TypeScript
- ✅ 使用 Cocos Creator 3.x API
- ✅ 使用新的 Component 装饰器系统
- ✅ 使用 Tween 动画系统替代 Action 系统
- ✅ 使用新的 AudioSource API
- ✅ 类型安全和更好的 IDE 支持
